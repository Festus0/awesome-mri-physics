# Awesome MRI Physics

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

A curated list of open-source tools, libraries, datasets, tutorials, and communities for **MRI physics**, **pulse sequence development**, **image reconstruction**, **MR spectroscopy**, **quantitative MRI**, and **neuroimaging workflows**.

This list is intended for **MRI physicists, imaging scientists, engineers, and researchers** working in both **preclinical and clinical MRI**.

# Pulse Sequence Development

- https://github.com/pulseq/pulseq — Vendor-neutral pulse sequence framework
- https://github.com/imr-framework/pypulseq — Python implementation of Pulseq
- https://github.com/pulseq/tutorials — Pulseq example sequences
- https://github.com/gpilab/gpi — Visual programming environment for MRI
- https://github.com/mriphysics/sequence-tools — Sequence design utilities
