# Contributing

Thanks for contributing to Awesome MRI Physics.

Please make sure additions are:
- relevant to MRI physics, reconstruction, spectroscopy, pulse sequence development, quantitative MRI, or adjacent imaging workflows
- open-source or openly accessible
- accompanied by a short neutral description
- not duplicates of existing entries

For link additions:
- use the project’s main repository or official homepage
- keep descriptions concise
- place the link in the most relevant section
