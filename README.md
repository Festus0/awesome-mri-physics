# Awesome MRI Physics

A curated list of open-source tools, libraries, datasets, tutorials, and communities for **MRI physics**, **pulse sequence development**, **image reconstruction**, **MR spectroscopy**, **quantitative MRI**, and **neuroimaging workflows**.

> This list is intended for researchers, students, MRI physicists, engineers, and imaging scientists working across preclinical and clinical MRI.

## Pulse Sequence Development

* [Pulseq](https://github.com/pulseq/pulseq) — Open framework for MR pulse sequence design and execution.
* [Pulseq Tutorials](https://github.com/pulseq/tutorials) — Practical examples for learning Pulseq.
* [PyPulseq](https://github.com/imr-framework/pypulseq) — Python tools for creating Pulseq-compatible sequences.
* [GPI Pulseq examples](https://github.com/gpilab) — Visual programming ecosystem used in some MR research workflows.

## Reconstruction

* [BART](https://github.com/mrirecon/bart) — Berkeley Advanced Reconstruction Toolbox for computational MRI.
* [Gadgetron](https://github.com/gadgetron/gadgetron) — Streaming reconstruction framework for MRI.
* [ISMRMRD](https://github.com/ismrmrd/ismrmrd) — Raw MR data format and tools for reconstruction workflows.
* [SigPy](https://github.com/mikgroup/sigpy) — Iterative methods and signal processing for imaging reconstruction.
* [Michigan Image Reconstruction Toolbox (MIRT)](https://github.com/JeffFessler/mirt) — Reconstruction algorithms and educational material.

## MR Spectroscopy

* [FSL-MRS](https://github.com/wtclarke/fsl_mrs) — End-to-end magnetic resonance spectroscopy analysis.
* [Gannet](https://github.com/markmikkelsen/Gannet) — MATLAB toolkit for edited 1H MRS analysis.
* [Osprey](https://github.com/schorschinho/osprey) — Open-source toolkit for MRS data processing and quantification.
* [spant](https://github.com/neuroconductor-devel/spant) — R package for MRS analysis.
* [FID-A](https://github.com/CIC-methods/FID-A) — Simulation and processing tools for MRS and RF pulse design.

## Quantitative MRI

* [qMRLab](https://github.com/qMRLab/qMRLab) — Quantitative MRI analysis framework.
* [hMRI Toolbox](https://github.com/hMRI-group/hMRI-toolbox) — Quantitative MRI workflows in SPM.
* [MRtrix3](https://github.com/MRtrix3/mrtrix3) — Widely used toolkit for diffusion MRI and related quantitative analyses.

## Diffusion MRI

* [DIPY](https://github.com/dipy/dipy) — Diffusion MRI analysis in Python.
* [MRtrix3](https://github.com/MRtrix3/mrtrix3) — Diffusion modelling, tractography, and processing.
* [QSIPrep](https://github.com/PennLINC/qsiprep) — Reproducible diffusion MRI preprocessing pipelines.

## fMRI and Neuroimaging

* [fMRIPrep](https://github.com/nipreps/fmriprep) — Standardized fMRI preprocessing.
* [NiBabel](https://github.com/nipy/nibabel) — Access to common neuroimaging file formats in Python.
* [Nilearn](https://github.com/nilearn/nilearn) — Statistical learning and visualization for neuroimaging.
* [FreeSurfer](https://github.com/freesurfer/freesurfer) — Structural neuroimaging analysis.
* [AFNI](https://github.com/afni/afni) — Analysis and visualization for functional MRI.
* [FSL](https://git.fmrib.ox.ac.uk/fsl) — Neuroimaging analysis library and tools.

## Data Conversion and Standards

* [dcm2niix](https://github.com/rordenlab/dcm2niix) — DICOM to NIfTI conversion.
* [BIDS](https://github.com/bids-standard/bids-specification) — Brain Imaging Data Structure standard.
* [heudiconv](https://github.com/nipy/heudiconv) — DICOM conversion and BIDS organization.
* [BIDS Validator](https://github.com/bids-standard/bids-validator) — Validation for BIDS datasets.

## Visualization and Analysis

* [3D Slicer](https://github.com/Slicer/Slicer) — Medical image visualization and analysis platform.
* [ITK](https://github.com/InsightSoftwareConsortium/ITK) — Toolkit for medical image analysis.
* [SimpleITK](https://github.com/SimpleITK/SimpleITK) — Simplified interface to ITK.
* [napari](https://github.com/napari/napari) — N-dimensional image viewer.

## Phantoms, Simulation, and Education

* [JEMRIS](https://github.com/JEMRIS/jemris) — MRI simulation platform.
* [Bloch Solver](https://github.com/bretglun/BlochSolver) — GPU-accelerated Bloch simulation.
* [MRzero](https://github.com/MRsources/MRzero-Core) — MRI simulation and education framework.
* [Awesome Neuroimaging](https://github.com/NPACore/awesome-neuroimaging) — Broad neuroimaging resource collection.

## Datasets

* [fastMRI](https://github.com/facebookresearch/fastMRI) — Accelerated MRI dataset and benchmarks.
* [OpenNeuro](https://github.com/OpenNeuroOrg/openneuro) — Shared neuroimaging datasets.
* [IXI Dataset](https://brain-development.org/ixi-dataset/) — Public structural MRI dataset.
* [Human Connectome Project](https://www.humanconnectome.org/) — Large-scale human neuroimaging resource.

## Communities and Learning

* [ISMRM](https://www.ismrm.org/) — International Society for Magnetic Resonance in Medicine.
* [OHBM](https://www.humanbrainmapping.org/) — Organization for Human Brain Mapping.
* [NeuroStars](https://neurostars.org/) — Community Q&A for neuroimaging.
* [Pulseq Tutorials](https://github.com/pulseq/tutorials) — Good starting point for sequence development.

## Contributing

Contributions are welcome. Please open a pull request to:

* add a tool or library,
* improve a description,
* fix a broken link,
* suggest a new category, or
* add educational resources and example datasets.

## Contribution guidelines

Please ensure that additions are:

1. Relevant to MRI physics, MRI engineering, MRS, reconstruction, pulse sequence programming, or closely related imaging workflows.
2. Open-source or openly accessible.
3. Actively useful to researchers, students, or developers.
4. Added with a short, neutral description.
