# Contributing Guidelines

Thank you for your interest in contributing to **Awesome MRI Physics**!

## Allowed contributions

- Open-source MRI tools, datasets, and tutorials
- Updates to existing entries
- New categories where clearly justified

## Not allowed

- Closed-source or commercial-only tools as standalone entries
- Personal blogs as the only reference

## Pull request process

1. Fork this repository.
2. Create a branch for your change.
3. Add or update the entry in `README.md` or related files.
4. Include a short, neutral description.
5. Link to the official project page or repository.
6. Fill in the pull request template.

## Suggested labels

- `type: addition`
- `type: maintenance`
- `type: question`
- `type: discussion`
- `good first issue`

## Starter issues

1. Add Bruker conversion tools
2. Add low-field MRI resources
3. Create notebooks landing page
4. Add sequence design tutorials
5. Add dataset usage notes
6. Add hardware resources
7. Add advanced reconstruction examples
8. Add MRS and qMRI data generators
9. Add community events calendar
10. Improve license table
